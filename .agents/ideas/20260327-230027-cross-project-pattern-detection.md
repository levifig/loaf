# Cross-project pattern detection

## Nugget

Agent notices the same correction or preference appearing across N projects and suggests promoting it to global config or personal knowledge. Automated learning from repeated feedback.

## Problem/Opportunity

Users repeat the same corrections across projects ("don't use mocks for DB tests", "prefer single bundled PRs"). These preferences are captured per-project in auto-memory but never promoted to a durable, cross-project location. The agent should detect the pattern and suggest promotion.

## Initial Context

Depends on personal knowledge base infrastructure existing first. Related to Mem0/OpenMemory territory but scoped to coding preferences, not general memory.
