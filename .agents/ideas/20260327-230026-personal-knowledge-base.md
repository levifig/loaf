# Personal knowledge base — portable coding personality

## Nugget

Atomic, version-controlled knowledge files that represent your coding personality — preferences, patterns, domain expertise — as a portable artifact that travels across projects.

## Problem/Opportunity

Project knowledge lives in project repos. But personal knowledge (coding style, tool preferences, domain expertise, recurring corrections) is scattered across auto-memory, PROFILE.md, and CLAUDE.md files. A structured personal knowledge base would let agents calibrate to you from day one on any project.

## Initial Context

Distinct from project knowledge (which is per-repo). Should integrate with QMD for retrieval. Location TBD — should not be project-specific.
